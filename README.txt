Кот и Код предстваляет вашему вниманию декстопное приложение moscowtraffic <<
moscowtraffic предназначен для расчёта трафика в зависимости от введённых в эксплуатацию жилых или нежилых зданий <<
Все входные данные были использованы для получения максимально информативного и наглядного рузультата, чтобы принцип работы приложения был понятен интуитивно <<
Приложение было написано в программе Visual Studio с применением технологий Windows Forms на языке программирования C#, в репозитории представлена корневая папка приложения <<
Для запуска .exe файла пройдите по следующему пути: moscowtraffic/bin/Debug/moscowtraffic.exe <<

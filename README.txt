Кот и Код предстваляет вашему вниманию декстопное приложение moscowtraffic <<
moscowtraffic предназначен для рассчёта трафика в зависимости от введённых в эксплуатацию жилых или нежилых зданий <<
Все входные данные были использованы для получения максимально информативного и наглядного рузельтата, чтобы принцип работы приложения был понятен интуитивно <<
Для запуска .exe файла пройдите по следующему пути: moscowtraffic/bin/Debug/moscowtraffic.exe <<
